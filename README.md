# SFL Amazon Laptop Search

**© 2026 [sjd_datascapes] | Simon James Drury**

## Overview

Systemic Functional Linguistics-powered review analysis for Amazon laptop search across **6 languages** (ES/EN/FR/IT/DE/PT).

### Key Features

- **Multi-language SFL Analysis**: Analyzes Amazon reviews in Spanish, English, French, Italian, German, and Portuguese
- **Three Metafunctions**:
  - **Ideational**: Process types, participants (RAM, performance, expandability)
  - **Interpersonal**: Evaluative language, modality, recommendation strength
  - **Textual**: Review coherence, credibility, information structure
- **Intelligent Ranking**: Combines price, Bayesian-adjusted ratings, and SFL quality scores

### Search Constraints

- RAM: ≥32 GB (minimum)
- Screen: ≥14 inches (minimum)
- Storage: ≥512 GB SSD (minimum)
- Memory: Expandable (preferred)

### Amazon Sites Supported

🇪🇸 Amazon.es | 🇬🇧 Amazon.co.uk | 🇫🇷 Amazon.fr | 🇮🇹 Amazon.it | 🇩🇪 Amazon.de | 🇵🇹 Amazon.pt

## Technical Details

### SFL Framework

Based on M.A.K. Halliday's Systemic Functional Linguistics:

- **Field-Tenor-Mode** analysis of review context
- **Transitivity** analysis for experiential meaning
- **Appraisal Theory** for interpersonal evaluation
- **Theme-Rheme** structure for textual organization

### Ranking Algorithm

1. **Price** (within spec constraints)
2. **Bayesian-adjusted rating** (accounts for review volume statistical significance)
3. **SFL Review Quality Score** (linguistic depth, relevance, credibility)
